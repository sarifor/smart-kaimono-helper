## 개선

* [ V ] trend, preference 내용 보여주는 거 삭제
* [ V ] 사용 금액 범위는 90%~110%를 허용
* [ V ] 추천되는 상품 풀이 몇 상품으로 한정(요구르트, 하겐다즈 등)되는 걸 수정
* [ P ] shop.py 쇼핑·키워드 에이전트 시스템 프롬프트의 一人暮らし向け 문구 제거: '一人暮らし向け → 日本在住者向け + 少量・使い切りサイズを基本とし、一人から二人分の量を想定'으로 대체하면 동작은 유지하면서 표현을 중립화할 수 있음.
* [ P ] API 호출 '재시도 중', '성공', '실패' 로그 출력되게 하기
* [ P ] 가격 참고표 데이터를 LLM Deep Research 모드로 조사하여, prices.json 보강
* [ P ] `trend_kr.md` (한국어) 또는 `trend_jp.md` (일본어)를 직접 수정하더라도 7일이 지나면 웹 검색으로 자동 갱신되는 문제 수정
