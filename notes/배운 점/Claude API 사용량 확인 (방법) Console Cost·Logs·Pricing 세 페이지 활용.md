# 상세
Claude API의 사용량과 지출은 Anthropic의 **웹 콘솔**(`platform.claude.com`)에서 세 개의 페이지를 통해 확인한다. 각 페이지는 관점이 다르다. **Cost 페이지**는 *지출 집계*, **Logs 페이지**는 *개별 요청*, **Pricing 페이지**는 *단가 기준*을 제공한다. 이 세 페이지를 조합하면 "얼마 썼는지 → 어디서 썼는지 → 계산이 맞는지"를 순서대로 확인할 수 있다.

**Cost 페이지**는 기간(일·주·월)별 지출을 **모델·API 키·워크스페이스별**로 집계하고, 같은 Console의 Billing 영역에서 결제 수단과 **Usage limits**(월 한도)를 설정한다. **Logs 페이지**는 개별 요청을 **모델명·입출력 토큰·타임스탬프·레이턴시·상태 코드**로 나열해 모델·API 키·시간대로 필터링할 수 있다. **Pricing 페이지**는 모델별 MTok 단가(Opus \$5/\$25, Sonnet \$3/\$15, Haiku \$1/\$5), 캐싱 배수(쓰기 1.25x~2x, 읽기 0.1x), Batch API 50% 할인, 서버툴 과금 등 계산 근거를 제공한다.

빠르게 지출 추세를 보고 싶으면 **Cost**, 특정 호출의 토큰·지연 문제를 보고 싶으면 **Logs**, 계산이 맞는지 검증하려면 **Pricing**을 쓴다. Opus 4.7은 **새 토크나이저**로 같은 텍스트에 **토큰 +35%**까지 소비할 수 있어, Cost에서 구세대 감각으로 예상하면 과소 추정 위험이 있다.

```
검증 보고:
- Pricing 페이지는 WebFetch로 직접 확인 (모델 단가·캐싱·Batch·서버툴·토크나이저 +35% 경고 원문 대조)
- Cost·Logs 페이지는 로그인 필요로 WebFetch 불가 — 설명은 Console 일반 UI 관행과 Pricing 문서의 "Usage tracking available in the Claude Console" 기술 근거
- 실제 UI 세부 라벨·메뉴 구조와 차이 시 업데이트 필요
```

## 시나리오 (비유)
월요일 아침 **체크카드 명세서(Cost 페이지)** 를 연다. 지난 주 총액과 가장 많이 쓴 가게(모델)가 한눈에 보인다. 지출이 이상하게 많으면 **영수증 꾸러미(Logs 페이지)** 에서 한 장씩 뒤적여 범인 호출을 찾는다. "이 숫자가 맞나?" 의심스러우면 **메뉴판(Pricing 페이지)** 의 단가와 곱해 본다. 세 페이지 역할 분담: **얼마 썼나 → 어디서 썼나 → 계산이 맞나**.

## 실전 체크리스트 (최대 3개)
- [ ] 주 1회 Cost 페이지 확인 + Billing의 `Usage limits`로 월 상한 설정.
- [ ] Cost에 이상 급증 발견 시 같은 기간을 Logs에서 필터링해 원인 호출 특정.
- [ ] Pricing 단가로 수동 검증: `(입력×입력단가 + 출력×출력단가) / 1,000,000`.

## 함정 (2~3개)
- Cost 집계는 실시간이 아님 — 방금 호출은 즉시 반영되지 않을 수 있음. 실시간 통제가 필요하면 코드 레벨 모니터링 병행.
- Opus 4.7은 새 토크나이저로 **토큰 +35%**까지 — Cost에서 Sonnet보다 예상 밖으로 비싸 보일 수 있음.
- Bedrock·Vertex·Foundry 경유 호출은 Pricing 페이지 수치와 다른 각 플랫폼 가격표를 따름.

## 복습 퀴즈 (3문항)
**Q1.** Cost·Logs·Pricing 세 페이지는 각각 어떤 질문에 답하는가?
**A1.** Cost = "얼마 썼나"(집계), Logs = "어디서 썼나"(개별 요청), Pricing = "계산이 맞나"(단가 근거).

**Q2.** 지출이 이상하게 많은 주를 발견했다. 원인을 좁히려면 어느 페이지로 먼저 이동해야 하는가?
**A2.** **Logs 페이지**. 같은 기간·모델·API 키로 필터링해 범인 호출을 특정.

**Q3.** Opus 4.7로 입력 50,000 / 출력 15,000 토큰을 1회 호출하면 대략 얼마인가?
**A3.** `50,000 × \$5 / 1,000,000 + 15,000 × \$25 / 1,000,000 = \$0.25 + \$0.375 = **\$0.625**`.
