## 목표

smart-kaimono-helper README 문서에 첨부할 영상 작성.<br/>
공감을 먼저 유발하고 앱을 소개하는 구조.

## 타깃

- 회사 일 등으로 집안일이 버거운 1인 가구 및 가족.

## 전체 구성

**총 길이: 한국어 버전 1분 2초, 일본어 버전 58초**

| # | 파트 | 방식 | 자막 / 오버레이 |
|---|---|---|---|
| 1 | 냉장고 안을 보기 + 뭔가 떠올림 | AI 애니메이션 (귀여운 레트로 픽셀 게임 스타일) | 없음 (표정으로 전달) |
| 2 | 터미널 — 언어 선택 | 실제 터미널 화면 녹화 | 없음 |
| 3 | 터미널 — 예산·키워드 입력 | 실제 터미널 화면 녹화 | 없음 |
| 4 | 터미널 — 결과 출력 | 실제 터미널 화면 녹화 | 없음 |
| 5 | 만족하는 모습 | AI 애니메이션 (귀여운 레트로 픽셀 게임 스타일) | 없음 (표정으로 전달) |

배경음: 루프형 lo-fi hip hop BGM, 레트로 게임 감성.

## 사용 툴

| 용도 | 툴 | 비고 |
|---|---|---|
| AI 애니메이션 작성 | **Google Flow** (Google AI Pro 플랜) | - |
| 터미널 화면 녹화 | **Windows 기본 캡처 도구** | - |
| 배경음 생성 | **Gemini 내 Lyria 3** (Google AI Pro 플랜) | - |
| 영상/배경음 편집 | **CapCut 데스크톱 버전** (무료) | 클립 연결, 배경음 삽입, 화면 일부 마스킹 등 |
| 배경음 편집 | **Audacity** (무료) | LUFS를 -16으로 조절 (귀에 부담 없는 음량) |
