## 紹介

日本のネットスーパーの買い物かごリストをAIが代わりに作成するCLIツールです。<br/>
予算を入力すると、トレンドや好みを反映した買い物かごを作成し、栄養バランスまでチェックします。<br/>
対象ユーザーは日本在住の韓国人・日本人・その他の日本語ユーザーです。韓国語・日本語の両方に対応しています。

## ストーリーで見る動作の流れ

月曜の夜、冷蔵庫を開けたら卵だけ。何を買えばいいか考えること自体が面倒だ。<br/>
`python shop.py`を実行して、予算とキーワードを入力するだけで終わる。<br/>
AIがトレンド・好み・栄養バランスを考えて買い物かごリストを作ってくれる。<br/>
そのリストを持ってネットスーパーにアクセスし、あとはかごに入れるだけでいい。

## 主な機能

| 機能 | 説明 |
|---|---|
| 🔑 キーワード提案 | 今日の日付・季節・トレンドを反映したショッピングキーワードを7つ提案 |
| 🛒 買い物かご生成 | 予算±10%以内に収めた商品リストを生成 |
| 🥗 健康チェック | 栄養バランスをもとに追加・削除・減量を提案 |
| 📄 結果保存 | `carts/`フォルダにMarkdownファイルとして自動保存 |
| 🌐 トレンド更新 | 日本のスーパーのトレンドをウェブ検索で7日ごとに自動更新 |
| 🇰🇷🇯🇵 二言語対応 | 韓国語・日本語を選択可能 |

## インストールと使い方

### 前提条件

- Pythonがインストールされている必要があります。
- [Anthropicコンソール](https://console.anthropic.com)でAPIキーを取得してください。

### インストール

1. ターミナル(cmd / PowerShell)を開き、プロジェクトを保存したいパスに移動します。
   ```
   cd 任意のパス
   ```
2. このリポジトリをクローンします。
   ```
   git clone https://github.com/sarifor/smart-kaimono-helper.git
   ```
3. 依存パッケージをインストールします。
   ```
   pip install -r requirements.txt
   ```
4. プロジェクトルートに`.env`ファイルを作成し、APIキーを入力します。
   ```
   ANTHROPIC_API_KEY=sk-ant-...
   ```

### 使い方

1. ターミナル(cmd / PowerShell)を開き、プロジェクトフォルダに移動します。
   ```
   cd smart-kaimono-helper
   ```
2. アプリを起動します。
   ```
   python shop.py
   ```
3. 言語を選択します (한국어 / 日本語)
4. 予算を入力します (円、税抜き、送料別)
5. 提案されたキーワードから選ぶか、直接入力します
6. 買い物かごリストが生成され、`carts/`フォルダに保存されます (アプリ自動終了)
