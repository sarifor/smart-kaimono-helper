## TL;DR

**장바구니 1개 생성 ≈ 약 $0.08 (약 12엔)**

자판기 음료 1병의 8% 수준이에요. 매일 한 번씩 한 달 내내 돌려도 $2.5 안 넘어요.

---

## 토큰이 뭔가요

LLM(Claude 같은 AI)은 글을 **"토큰" 단위**로 읽고 써요. 토큰은 글자보다 살짝 큰 덩어리예요.

대략적으로:

- 영어: 1 단어 ≈ 1 토큰
- 한국어·일본어: 1~2 글자 ≈ 1 토큰
- `안녕하세요` → 약 5~7 토큰
- `納豆 3パック` → 약 6~8 토큰

문장이 길수록 토큰을 더 많이 써요.

## 입력 vs 출력

한 번의 API 호출에는 두 종류의 토큰이 있어요:

- **입력 (input)**: AI한테 보낸 내용 전체. 시스템 지시, 가격 참고표, 사용자 요청 등이 다 포함돼요.
- **출력 (output)**: AI가 뱉은 답변.

**출력이 입력보다 5배 비싸요.** 글 읽는 것보다 **생성**하는 게 컴퓨팅 자원이 훨씬 많이 들기 때문이에요.

Claude Sonnet 4.6 가격:

| 종류 | 가격 (1,000,000 토큰당) |
|---|---|
| 입력 | **$3.00** |
| 출력 | **$15.00** |

상세: [Claude API Docs/ Pricing](https://platform.claude.com/docs/en/about-claude/pricing)

## 실제 측정 예시

### 조건 & 토큰 사용량

장바구니를 **1회 생성**한 실제 데이터예요.<br/>
API는 총 4회 호출했어요. (재시도 1회 추정)

- 언어 모드: 한국어
- 예산: 8,000엔
- 키워드: 이번 주 트렌드 식품 (이미 작성되어 있는 트렌드 파일 사용)

| 시각 (GMT+9) | 호출 | 입력 토큰 | 출력 토큰 |
|---|---|---:|---:|
| 15:36:02 | 🔑 키워드 생성 | 482 | 104 |
| 15:36:34 | 🛒 쇼핑 에이전트 | 2,834 | 1,898 |
| 15:36:42 | 🥗 건강 에이전트 (1차) | 3,627 | 532 |
| 15:36:48 | 🔁 건강 에이전트 (재시도) | 4,517 | 468 |
| **합계** | | **11,460** | **3,002** |

### 각 호출이 무슨 뜻인가요

**🔑 키워드 생성 (가장 가볍)**
- 오늘 날짜·요일·계절을 기반으로 "오늘의 추천 키워드 3개"를 만들어요.
- 입력이 짧은 이유: 시스템 지시 + 트렌드 파일 요약만 보내거든요.
- 출력이 짧은 이유: 키워드 3개짜리 JSON만 뱉어요.

**🛒 쇼핑 에이전트 (출력이 가장 무거움)**
- 실제로 상품 리스트를 만드는 메인 호출이에요.
- 입력이 큰 이유: **가격 참고표(82개 품목)를 통째로 프롬프트에 넣기 때문**이에요.
- 출력이 큰 이유: 상품마다 `일본어명/한국어명/카테고리/가격/출처` 필드가 있는 JSON을 뱉어요.

**🥗 건강 에이전트**
- 쇼핑 에이전트가 만든 리스트를 보고 영양 균형을 확인해요.
- 쇼핑 에이전트와 같이 가격 참고표를 받아서 입력이 비슷한 크기예요.
- 출력은 코멘트 + 추가/삭제 제안 정도라 짧아요.

**🔁 건강 에이전트 재시도 (발생 시에만)**
- 건강 에이전트의 comment와 실제 행동(remove/reduce)이 불일치할 때 1회 재시도해요.
- 재시도 시 원본 상품명 목록을 추가로 보내기 때문에 입력이 1차보다 더 커져요.
- 재시도가 없으면 총 3회 호출로 끝나요.

## 비용 계산

### 입력 비용
```
11,460개 × ($3.00 ÷ 1,000,000개) = $0.03438
```

### 출력 비용
```
3,002개 × ($15.00 ÷ 1,000,000개) = $0.04503
```

### 총 비용
```
$0.03438 + $0.04503 ≈ $0.079
```

약 **8센트**, 엔으로 환산하면 **약 12엔** 정도예요.

> 재시도 없이 3회 호출로 끝난 경우는 약 $0.03~$0.05 수준으로 낮아져요.

## 비용을 더 줄이고 싶다면

실제로 이 앱의 **쇼핑 에이전트만 Sonnet 4.6**이 필요하고, 키워드 생성과 건강 에이전트는 **Haiku 4.5**로도 충분해요.

| 모델 | 입력 (1M 토큰당) | 출력 (1M 토큰당) |
|---|---:|---:|
| Claude Sonnet 4.6 | $3.00 | $15.00 |
| Claude Haiku 4.5 | **$1.00** | **$5.00** |

Haiku 4.5는 **1/3 가격**이라, 키워드·건강 에이전트를 Haiku로 바꾸면 전체 비용이 약 **절반 이하**로 내려가요. (품질 저하는 거의 없어요 — 단순한 작업이라서.)
