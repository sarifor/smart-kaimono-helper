# smart-kaimono-helper

일본 거주자용 넷슈퍼 장바구니 생성 CLI. 한국어/일본어 지원.
Anthropic API 기반 에이전트 체인 (키워드 → 쇼핑 → 건강).

상세 구조·동작·이력은 `HOW-IT-WORKS.md` 참고.

## 환경
- Windows + PowerShell
- Python, `.env`에 `ANTHROPIC_API_KEY`

## 상시 규칙

- **`shop.py` 수정 전 사용자 승인 필수**. 버그 발견해도 고치지 말고 보고만.
- **언어 관련 작업은 한국어/일본어 양쪽 대칭으로**. 한쪽만 구현 금지.
- **언어 분리 엄격**: 한국어 모드에 일본어 섞지 말 것, 반대도 마찬가지.
- **API 호출은 비용 발생**. 기본 테스트는 mock으로, 실제 호출 테스트는 `@pytest.mark.smoke`로 격리해서 `pytest -m smoke`로만 실행되게. 스모크 테스트는 한국어/일본어 각 1개씩 (총 2개) 필수.
- **가격은 전부 세전**, 배송비 별도. 이 컨벤션 유지.
- 작업 전 관련 파일 먼저 읽고 추측 금지.
- **코드 수정 후 검증 필수**:
  - `tests/` 존재 시 → `pytest -v`로 리그레션 확인
  - `tests/` 미존재 시 → 최소한 `python -c "import shop"`으로 구문 검증
  - 실패 시 테스트 버그인지 앱 버그인지 판단 후 보고
- **작업 중 배운 것은 `HOW-IT-WORKS.md`에 자동 반영**. 아키텍처, 버그 해결, 교훈, 주의사항 등.
